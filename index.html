Turtle Adventures

Python, Flask, SQLite, JavaScript, Bootstrap, HTML5, CSS3

Turtle Adventures is a website application that helps users disconnect from their devices and connect with nature by finding information about parks, getting outside, and going for a walk.

Turtle Adventures is great way to explore NYC parks. This application allows users to browse parks by borough and access more information stored in the NYC Parks Properties API. Users create an account by selecting a username and password which gives them access to the website. The app returns all NYC parks which can then be sorted by borough. Users may click on a park name to access its website for more information.

Nature Walk

JavaScript, Node, Express, Bootstrap, HTML5, CSS3

Nature Walk is a web application developed for children to encourage them to explore the outdoors.

JavaScript requests data from the Simple Wikipedia API which delivers a response that is parsed to extract the information needed to fulfill the project requirements (website functionality) and user stories (what the user should be able to do when interacting with an application or a website). JavaScript accesses this data and manipulates the DOM to dynamically populate the screen with the selected content. A custom Google Search field is included for users to expand their learning.

Nature Walk allows children to learn about life forms they might find in the woods. My hope is that Nature Walk will:

  • Inspire children to ask their parents and family members to go on adventures and explore the outdoors.
  • Promote care and respect for the environment.
  • Reignite curiosity in the adults who join the adventure and remind them how much fun it is to connect with nature.
Spiral Twists

HTML5, CSS3, JavaScript, Bootstrap

Spiral Twists is a digital gallery of my work as a potter.

Starting with a mobile first approach, all images were scaled down to improve performance and media queries were added to adjust the layout for different screen sizes. The links change appearance when a user hovers over them to indicate an interaction is possible. CSS3 transition and transform properties animate the gallery categories. JavaScript controls the display of the navigation bar by responding to the user when the menu button is clicked.
